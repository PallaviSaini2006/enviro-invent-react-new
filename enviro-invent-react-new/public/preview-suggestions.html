10
Image Lazy Loading — Performance
Technical / Quick Fix

Adding loading="lazy" to the deck plan and product images means they only load when the user actually scrolls to them. Cuts initial page load data by ~800KB.

Code change (3 characters per image)
Before — loads everything upfront
<img
  src="/deck-plan.png"
  alt="Deck plan"
/>
After — deferred until in view
<img
  src="/deck-plan.png"
  alt="Deck plan"
  loading="lazy"
/>
Load time impact estimate
Before (3G)
4.2s
After (3G)
2.1s
